Differences between var, let and const.

1. Var

The scope of a var variable is functional scope. It can be updated and re-declared into the scope.

2. Let

The scope of a let variable is block scope. It can be updated but cannot be re-declared into the scope.

3. Const

The scope of a const variable is block scope. It cannot be updated or re-declared into the scope.

Difference between map, foreach, filter and find.

1. ForEach

Foreach takes a callback function and run that callback function on each element of array one by one.

2. Map

Map like filter & foreach takes a callback and run it against every element on the array but whats makes it unique is it generate a new array based on your existing array.

3. Filter

Whenever you have to filter an array Javascript inbuilt method to filter your array is the right choice to use. Filter let you provide a callback for every element and returns a filtered array.

4. find

As the name already suggest reduce method of the array object is used to reduce the array to one single value.

Why we use template string.

Template literals are sometimes informally called template strings, because they are used most commonly for string interpolation (to create strings by doing substitution of placeholders).
